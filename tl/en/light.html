Installation

1

Install YourTelebot

Go to the GitHub project, select the latest release, and download it (Assets â†’ Source Code).

Extract the archive to any convenient location.

If you're reading this, then you've already completed point 1! Here you go!

2

Download Python

Visit the official Python website and install one of the latest versions.

Important: During installation, enable "Add Python to PATH." This is necessary for the program to automatically install required libraries.

In case of errors:

  • Try downloading an older version of Python.
  • If you already had Python installed, remove everything related to it.
  • Restart your computer.
  • If this doesn't help, search for a solution online or contact support!

FAQ

(Most Popular Questions)

Is it safe?

<

Yes, but you can significantly increase your security by following three simple steps:

  1. Download the application only from official sources (Install)
    This way, you can be sure that no malicious software is embedded in the application.
  2. Do not grant access to people you do not trust.
    Keep in mind that account owners whose IDs are added to the program have almost full access to your PC!
  3. Enable 2FA in Telegram (two-factor authentication)
    This will reduce the risk of your account being hacked, thereby increasing the application's security.
    To enable 2FA in Telegram:
    Settings > Privacy > Cloud Password.

If you have sufficient knowledge of Python or cybersecurity, the source code is available for you.

It shows an error. What should I do?

<

Usually, the error message comes with a solution. But since you're here, it seems that didn't help.

We offer you several solutions:

  1. Reinstall the program.
    This will help if you have modified it and are unsure what went wrong.
  2. Run the program through IDLE (Integrated Development Environment)
    If the application doesn't open at all:
    In the Start menu, find "IDLE (Python your version)". Launch it and open YourTelebot.pyw. After pressing F5, a console will appear, with the last line displaying the error.
    First, try to solve it yourself using the internet and AI tools.
    If you can't, contact support or reinstall the program.

Note that YourTelebot may not run in some development environments, such as VS Code. This happens because the compilers in these environments require full paths (C://Users//User...), while the program's code uses relative paths (data/images/image.jpg).
Therefore, we recommend running the application using pythonw.exe, python.exe, IDLE, or Sublime Text.

Who can control my PC?

<

Anyone who has access to the accounts whose IDs you specified in the "Users" menu.

We recommend checking your account for connected devices and not giving access to others.

You can see which devices your account is logged into inside Telegram: Settings â†’ Devices. Remove any that you donâ€™t use.

How does it work?

<

  1. You launch the program, insert the token and add users.
  2. Using the token, the application turns your computer into a server and activates the bot.
  3. When any command is sent, the bot checks whether the account is in the user list. If yes, it responds; if not, it ignores the request.

How to add multiple users?

<

Remember, by granting access to accounts that may be used by others, you are giving them full access to your computer.

To add a new user:

  1. Find out the account ID. You can do this using the /start command in your bot. This command is available to everyone.
  2. Add the obtained ID to the program by entering it in the "Users" section, each on a new line. This works as follows: 1 user = 1 line.
